Senior Thesis

I worked with Professor Kevin Chang and Nikita Spirin to help understand how we can search for structured information on in the unstructured data of the world wide web.

Specifically, we investigated how to retrieve relevant pages given entities as queries. We formalized the problem by creating a two step process, first iteratively querying an existing keyword search engine in the retrieval phase, then aggregating these results and reranking them to present a final set of results to the user in the ranking phase. A basic diagram of this process is shown on the right.

During the retrieval phase, we investigated strategies for keyword selection, query operators, and query order. Likewise, we used machine learning to develop an effective ranking strategy using various features for each document output from the retrieval phase.

Ultimately, our results demonstrate that we can reduce the number of queries necessary by carefully selecting keywords for each entity in domain-independent way, and that using only a simple machine learning strategy, our initial implementation is able to outperform the results provided by Google on our test entities.

CoMoTo · The Collaboration Modeling Toolkit

Since Fall 2010, I have been working with Cinda Heeren, Charlie Meyer, and Eric Shaffer to help develop software for CoMoTo. This software uses advanced software similarity analysis tools such as MOSS to help instructors detect cheating in several mainstream undergraduate classes. However, our team likewise uses the tool to study how students collaborate and how students learn in these classes.

Through our efforts to analyze collaboration in undergraduate classes, we have provided a useful tool to the department, and have gained a better understanding of how students learn and understand the concepts and projects at hand.


Deep Web Search Interfaces

Through Spring 2010, I studied deep web search interfaces, and developed a set of design principles for interfaces for any deep search engine. I compared the interfaces of several existing seep web search engines and used a bottom-up analysis of these interfaces to construct a set of generalized principles for deep web search interfaces.


P.U.R.E. · Promoting Undergraduate Research in Engineering

I actively promote research through my work with P.U.R.E., Promoting Undergraduate Research in Engineering, as the liason for the computer science department. We strive to involve undergraduates, particularly underclassmen, in undergraduate research by acting as an intermediary between undergraduate students, and graduate students and faculty, placing new researchers into active research projects each semester. Many of these students continue long term research with their mentors, and we believe this experience is a critical component of an undergraduate education.
